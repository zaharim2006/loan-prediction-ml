# 🏦 Loan Prediction ML Loyihasi

Bank mijozlarining ma'lumotlari asosida kredit tasdiqlanish yoki rad etilishini bashorat qiluvchi Machine Learning loyihasi.

## 📊 Dataset
[Kaggle: Loan Prediction Dataset](https://www.kaggle.com/datasets/ninzaami/loan-predication)

## 🔧 Ishlatilgan texnologiyalar
- Python, Pandas, NumPy
- Matplotlib, Seaborn (EDA)
- Scikit-learn (Logistic Regression, Decision Tree, Random Forest)
- XGBoost

## 📈 Bosqichlar
1. Data Cleaning — bo'sh qiymatlarni to'ldirish
2. EDA — grafiklar orqali tahlil
3. Feature Engineering — yangi ustunlar, log transformatsiya, encoding
4. Model Training — 4 ta model solishtirildi
5. Hyperparameter Tuning — GridSearchCV
6. Model baholash — Accuracy, Precision, Recall, F1-score

## 🏆 Natijalar

| Model | Accuracy | F1-score |
|---|---|---|
| Logistic Regression | 0.7886 | 0.8587 |
| Decision Tree | 0.6504 | 0.7296 |
| Random Forest | 0.7561 | 0.8333 |
| XGBoost | 0.7805 | 0.8421 |

## 🔑 Eng muhim omil
**Credit_History** (kredit tarixi) — kredit qaroriga eng kuchli ta'sir qiluvchi omil bo'lib chiqdi.
